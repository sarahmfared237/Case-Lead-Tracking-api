Simple Lead & Case Tracking API

A beginner-friendly Node.js + Express API that manages Leads and Cases using local JSON files instead of a database.
This project fulfills the requirements of a junior-level assignment involving CRUD operations, input validation, and file-based persistence.

Installation & Setup:

1. Install dependencies
npm install

2. Start the server
npm start

Server runs on:
http://localhost:3000
